# Fashion Retail SQL Analysis

Using SQL to analyze fashion/clothing retail sales data — what sells, when it sells, and who the best customers are — and turning it into business recommendations.

> **Status: In progress** — building queries and writing up findings.

## About
As someone who has run a clothing line, I'm digging into fashion retail data with SQL to practice real analyst work: answering business questions and translating them into what a business should actually do.

## Tools
- SQL (SQLite via DB Browser for SQLite)
- Dataset: fashion / clothing e-commerce sales data

## Business questions
1. Which product categories drive the most revenue?
2. Which months / seasons drive the most sales?
3. Who are the top 10 customers by total spend?
4. What is the average order value, and how does it trend over time?
5. Which categories get returned most? (if available)
6. Revenue by region / channel (if available)
7. (Stretch) Rank products within each category using a window function
